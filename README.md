# SIGACE 🎓

### Sistema Inteligente de Gestión Académica y Control de Estudios

**SIGACE** es una solución integral diseñada para la automatización de procesos administrativos y académicos en instituciones educativas. Permite centralizar la información de estudiantes, docentes y representantes bajo una arquitectura segura y escalable.

---

## 🚀 Funcionalidades Principales

- **Gestión de Inscripciones:** Proceso automatizado para el registro de alumnos en periodos académicos específicos.
- **Control de Calificaciones:** Registro de notas por lapsos, asignaturas y secciones.
- **Gestión de Usuarios:** Roles diferenciados para Administradores, Docentes, Estudiantes y Representantes.
- **Generación de Reportes:** Creación de boletas de calificaciones y listados de secciones en formato PDF.
- **Arquitectura de Datos:** Relación robusta entre grados, secciones y carga docente.

## 🛠️ Stack Tecnológico

El proyecto está construido con las herramientas más modernas del ecosistema Web:

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Estilos:** [Tailwind CSS](https://tailwindcss.com/)
- **Base de Datos:** [MySQL](https://www.mysql.com/)
- **ORM:** No aplica (backend API externo)
- **Lenguaje:** [JavaScript](https://www.typescriptlang.org/)
- **Autenticación:** JWT / sesión manejada por backend API
